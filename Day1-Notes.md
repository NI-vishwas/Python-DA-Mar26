# Python Notes

## Programming Fundamentals
- High level Programming Language(HL)
- Systems understand 1/0, HL's **English** to **1/0**
- necessary words forms an Instruction
- Two wauys to give instructions
  - Compilation - Take whole set of instructions (source code) -> do checking grammer (syntax check) -> convert to **1/0**; The tool doing conversion is **Compiler**; 1/0 -> run the executable file (on computer), Example: C/C++/Java...
  - Interpretation - Script with commands; Take each command; process it (do checking, convert to 1/0); execute it, Example: Python, javascript...
